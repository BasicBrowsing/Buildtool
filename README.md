## Bundling

*How to bundle the project to an addon.*

```sh
deno run                                \
    --allow-all                         \
    --unstable                          \
    --importmap=Buildtool/Imports.json  \
    Buildtool/Source/App.js
```

*This is a temporary way of executing the bundler.*
